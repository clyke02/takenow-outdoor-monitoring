## Hasil yang Diharapkan

Wireframe akan menampilkan:

- Login form yang centered dan professional
- Clear input fields untuk username dan password
- Login button yang prominent
- Header dan footer yang informatif
- Layout yang clean dan mudah dipahami

## Variasi (Opsional)

Bisa juga request 2 wireframe terpisah:

1. **Login Page - Normal State** (seperti prompt di atas)
2. **Login Page - Error State** (tambahkan error message di bawah form)
